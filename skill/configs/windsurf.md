# Folkline UI — Windsurf Rules
# Place in .windsurf/rules/folkline-ui.md

**Folkline UI v3.0.0** — Utility-first CSS framework with 2,734 utility classes and 30+ accessible components.

## Quick Start
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@realtoheed/folkline-ui@3/dist/folkline.min.css" />
```

## Guidelines
1. Use utility classes for spacing, layout, colors, typography, and styling
2. Use pre-built component classes for complex UI patterns (badge, card, modal, tabs, etc.)
3. Always include proper ARIA attributes for accessibility
4. Add `:focus-visible` support for keyboard navigation
5. Use `prefers-reduced-motion` media query for animations
6. Colors use OKLCH color space for consistent harmonies
7. Available breakpoints: sm(640px), md(768px), lg(1024px), xl(1280px), 2xl(1536px)
8. Components include: button, card, badge, spinner, skeleton, modal, tabs, accordion, dropdown, toast, switch, pagination, progress, avatar, tooltip, breadcrumb, table, menu, timeline, rating, carousel, stepper, stat, list group, faq, upload zone, compare, hero section, alert, navbar, form elements
