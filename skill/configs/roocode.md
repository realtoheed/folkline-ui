**Folkline UI v3.0.0** — Utility-first CSS framework.

## Installation
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@realtoheed/folkline-ui@3/dist/folkline.min.css" />
```

## Available Classes (2,734 total)
- Spacing: padding, margin, gap (0-10 rem)
- Typography: text-{xs..9xl}, font-{100..900}, alignment, transforms
- Colors: text/bg/border with {primary,secondary,success,danger,warning,info,light,dark}-{50..900}
- Layout: flex, grid, display, position, overflow
- Sizing: w, h, max-w, min-h (fractions, fixed, viewport)
- Effects: shadows, borders, radius, opacity, transforms, filters
- Motion: transitions, animations (fade-in, spin, pulse, bounce)
