# Folkline UI — Superflex Rules

Folkline UI v3.0.0 is a utility-first CSS framework with 2,734 utility classes and 30+ accessible components.

CDN: https://cdn.jsdelivr.net/npm/@realtoheed/folkline-ui@3/dist/folkline.min.css

Use utility classes for all styling. Use component classes for complex UI patterns. Always include ARIA attributes for accessibility.
